HTML, Where it Started and Where it is Going

Homepage

Intro

The world wide web exists and functions because of the evolution and development of the HTML language. It is the foundation to everything we see online. HTML is an extremely pivotal part of how we view the internet, it has grown and adapted to meet the needs of new technology and expectations of the new generation of users.

HTML stands for hyper text markup language and is the basic standard language for creation of readable web pages on the world wide web. In short, HTML tells the browser how to display messages by labeling different components like “this is a paragraph” or “this is a heading”. It is the basic structure that can be built upon. The language breaks this down by placing what is being posted using greater and lesser signs, such as <body> or <title>. These bracketed words are called tags, and you can start a paragraph with <p> (enter content) </p>.

A web browser will take that data and display the content on a web page, but instead of seeing bracketed letters, you see the paragraph. A web page can be more than just letters and that is where CSS and JavaScript come into play. CSS (cascading style sheets) brings the structure to life by adding color, fonts, layout, and spacing. JavaScript adds interactive behavior, like pop-ups, animations, or responding to clicks.

Key Terms

  • HTML: hyper text markup language used to structure web pages
  • CSS: cascading style sheets used for design, layout, and styling
  • JavaScript: adds behavior and interactivity to web pages
  • API: application programming interface used for data communication
