Key Concepts

Here are the most important HTML-related concepts:

  • Semantic HTML — using meaningful tags like <header>, <main>, <footer>
  • Responsive design — layouts that adjust to screen size
  • CSS separation — structure (HTML) vs style (CSS)
  • APIs — HTML5 APIs like Web Storage, Canvas, Geolocation
  • Accessibility — using alt text, ARIA labels, readable structure
