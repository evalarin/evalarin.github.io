HTML, Where it Started and Where it is Going

Key Concepts

Core Ideas

HTML is the structural foundation of the web, using tags to describe content and create readable pages.

CSS expands HTML by adding styling, spacing, layout control, colors, and design.

JavaScript brings behavior, interactivity, and dynamic content to modern sites.

Key Terms

  • Elements: Building blocks of HTML pages
  • Tags: Opening/closing markers defining structure
  • Attributes: Extra information added to elements
  • Semantic HTML: Tags with meaning (header, main, nav)
  • API: Interfaces that allow apps to communicate
